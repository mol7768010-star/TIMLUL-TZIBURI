# מערכת תמלול אוטומטית ל-Call2All (Python)

מערכת זו מאפשרת לקבל קובץ שמע משרתי Call2All באמצעות בקשת GET, לתמלל אותו לעברית באמצעות בינה מלאכותית, ולשמור את התוצאה חזרה לשרת כקובץ טקסט (`.tts`).

---

## 🛠 פרמטרי ה-GET הנתמכים

כדי להפעיל את התמלול, עליך לשלוח בקשת `GET` לכתובת השרת שלך בפורמט הבא:

| פרמטר | חובה | תיאור |
| :--- | :--- | :--- |
| **token** | כן | הטוקן (API Key) של המערכת שלך ב-Call2All. |
| **K** | כן | הנתיב של הקובץ להורדה (כל מה שבא אחרי `ivr2:`). לדוגמה: `1/M1000.wav`. |
| **N** | לא | נתיב חלופי לשמירת קובץ הטקסט שיווצר. |

---

## 🚀 אופן פעולת המערכת

1. **הורדה:** המערכת פונה לכתובת ההורדה של Call2All ומושכת את הקובץ המצוין בפרמטר `K`.
2. **עיבוד:** הקובץ מעובד ומפוענח לטקסט בעברית (Speech-to-Text).
3. **שמירה (Upload):** * אם **לא** צורף הפרמטר `N`: הקובץ יישמר באותו נתיב בשינוי סיומת ל-`.tts`.
    * אם **כן** צורף הפרמטר `N`: הקובץ יישמר בתיקייה `N` עם שם הקובץ המקורי (בסיומת `.tts`).
4. **תגובה:** השרת מחזיר את הטקסט שתומלל כתשובה ישירה (Plain Text).
